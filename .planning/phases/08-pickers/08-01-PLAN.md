The drag pattern (from AeroHsvColorSquare.kt, Phase 7) — replicate this structure exactly:
```kotlin
.pointerInput(enabled, valueRange) {
    awaitPointerEventScope {
        while (true) {
            val down = awaitFirstDown(requireUnconsumed = false)
            // pick nearest thumb, fire onValueChange, down.consume()
            while (true) {
                val event = awaitPointerEvent()
                val change = event.changes.firstOrNull() ?: break
                if (!change.pressed) break
                // compute new value from change.position.x, fire onValueChange, change.consume()
            }
        }
    }
}
```
