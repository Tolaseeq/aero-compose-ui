AeroColorPicker.kt — ALREADY renders `AeroHueSlider(hue, onHueChange=...)` in the first Row right of
`AeroHsvColorSquare`. AeroHueSlider is a 24x256dp Canvas. The square is 256x256dp. So the hue slider
SHOULD be visible. F12 ("not findable") most likely cause: the panel is sized by an external/full
width that the Row distributes so the 24dp slider is pushed off, OR the panel column is wider than the
window so the slider clips. The panel uses `.glassPanel(cornerRadius=8.dp).padding(12.dp)` with NO width
constraint, so inside the Popup it takes intrinsic width — but if the showcase or popup stretches it,
the slider can be cut. Investigate: ensure the HSV-square Row gives the hue slider its 24dp and that the
panel width is intrinsic (wrap content), not full-window.

AeroColorPickerButton.kt — the Popup hosts `AeroColorPicker(...)` directly (NOT inside the date pickers'
`PickerPopupContainer` two-layer glass surface). AeroColorPicker provides its own `.glassPanel`. F10 wants
the SAME glass bg + rounded corners as the other pickers AND intrinsic width. PickerPopupContainer is the
shared W11-02 surface (import com.mordred.aero.components.pickers.internal.PickerPopupContainer). Wrapping
AeroColorPicker in PickerPopupContainer would double-glass; instead give AeroColorPicker an intrinsic
`Modifier.width(IntrinsicSize.Max)` or a fixed content width so it cannot stretch, and keep its glassPanel.

AeroColorPicker before/after preview bar (Row of two Boxes, beforeColor | currentColor) has NO labels —
F10 "left half stays default colour regardless of selection, purpose unclear". Add small labels/captions.

AeroAccordion.kt — the inter-section divider is `Box(Modifier.fillMaxWidth().height(1.dp).background(...))`
placed between sections. Each section header uses `.glassPanel(cornerRadius = 8.dp)` (rounded). The
full-width 1dp divider therefore extends to the rounded corners' horizontal extent and visually pokes
past the rounded bg edge (F11). Fix: inset the divider horizontally (e.g. `padding(horizontal = 8.dp)`)
so it sits within the rounded background's straight edge.
