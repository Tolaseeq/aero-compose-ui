## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] Valkyrie generated defaultWidth=256.dp instead of 24.dp**
- **Found during:** Task 4 (inspection of generated files)
- **Issue:** Valkyrie sets defaultWidth/defaultHeight from SVG viewBox size when SVG lacks explicit `width` attr. Phosphor Raw SVGs use viewBox="0 0 256 256" but no explicit `width="24"`.
- **Fix:** Applied `sed` to all 7 generated files to replace `256.dp` with `24.dp` for defaultWidth/defaultHeight.
- **Files modified:** All 7 `library/src/main/kotlin/com/mordred/aero/icons/internal/*.kt`
- **Verification:** `grep defaultWidth` confirms 24.dp in all files; compileKotlin passes.
- **Committed in:** `1764a8a` (Task 4 commit)

**2. [Rule 1 - Bug] Missing import com.mordred.aero.icons.AeroIcons in internal/*.kt**
- **Found during:** Task 5 (compileKotlin run)
- **Issue:** Valkyrie generates extension properties `public val AeroIcons.X: ImageVector` without importing `AeroIcons` from parent package. Compile fails with "Unresolved reference 'AeroIcons'".
- **Fix:** Added `import com.mordred.aero.icons.AeroIcons` to all 7 internal/*.kt files.
- **Files modified:** All 7 `library/src/main/kotlin/com/mordred/aero/icons/internal/*.kt`
- **Verification:** compileKotlin BUILD SUCCESSFUL.
- **Committed in:** `7ecec53` (Task 5 commit)

**3. [Rule 1 - Bug] X.kt fill=SolidColor(Color.Black) on stroke-only line paths**

## Valkyrie Output Shape (Resolved Open Question)

**Shape A confirmed.** Valkyrie 1.1.1 with `--output-format backing-property --iconpack-name AeroIcons` generates:

```kotlin
// internal/X.kt
package com.mordred.aero.icons.`internal`

import com.mordred.aero.icons.AeroIcons  // must be added post-generation
import androidx.compose.ui.graphics.vector.ImageVector
// ...

public val AeroIcons.X: ImageVector
    get() {
        if (_X != null) { return _X!! }
        _X = ImageVector.Builder(name = "X", defaultWidth = 24.dp, ..., viewportWidth = 256f, ...).apply {
            path(stroke = SolidColor(Color.Black), strokeLineWidth = 16f, strokeLineCap = StrokeCap.Round, ...) {
                moveTo(200f, 56f)
                lineTo(56f, 200f)
            }
        }.build()
        return _X!!
    }

@Suppress("ObjectPropertyName")
private var _X: ImageVector? = null
```

**AeroIcons.kt** is a hand-authored facade (`public object AeroIcons {}`) — Valkyrie does NOT generate it.
