Determinate implementation (concrete):
- Box outer `Modifier.fillMaxWidth().height(height + if (showPercent) 18.dp else 0.dp)`.
- Inside Column:
  - Track: `Box(Modifier.fillMaxWidth().height(height).background(colors.surface, RoundedCornerShape(50)))` then inner fill `Box(Modifier.fillMaxWidth(progress.coerceIn(0f, 1f)).height(height).background(colors.primary, RoundedCornerShape(50)))`.
  - if showPercent: `Text("${(progress * 100).toInt()}%", color = colors.labelText, style = AeroTheme.typography.bodySmall)` aligned end.

Indeterminate implementation (concrete):
- Use `rememberInfiniteTransition(label = "indeterminate")`.
- Animate a Float `shimmerX` from -0.3f to 1.0f with `infiniteRepeatable(animation = tween(1500, easing = LinearEasing), repeatMode = RepeatMode.Restart)`.
- Box outer fillMaxWidth().height(height).clip(RoundedCornerShape(50)).background(colors.surface).
- Inside, a moving 30%-wide Box: `Box(Modifier.fillMaxWidth(0.3f).height(height).offset { IntOffset((shimmerX.value * boxWidth).roundToInt(), 0) }.background(colors.primary))`.
- For simplicity use `Modifier.fillMaxHeight().fillMaxWidth(0.3f)` plus `align` based on `shimmerX`. Capture container width via `BoxWithConstraints` and convert `shimmerX * maxWidth` to dp offset. Concrete shape:
```kotlin
val transition = rememberInfiniteTransition(label = "indeterminate")
val shimmer by transition.animateFloat(
    initialValue = -0.3f,
    targetValue = 1.0f,
    animationSpec = infiniteRepeatable(animation = tween(1500, easing = LinearEasing), repeatMode = RepeatMode.Restart),
    label = "shimmerX"
)
BoxWithConstraints(modifier.fillMaxWidth().height(height).clip(RoundedCornerShape(50)).background(AeroTheme.colors.surface)) {
    val offsetX = with(LocalDensity.current) { (shimmer * maxWidth.toPx()).toDp() }
    Box(
        Modifier
            .fillMaxWidth(0.3f)
            .fillMaxHeight()
            .offset(x = offsetX)
            .background(AeroTheme.colors.primary)
    )
}
```

Note on default-arg: `AeroTheme.colors.primary` cannot be a default value at the top level (not @Composable context). Workaround: declare overload pair OR use `Color.Unspecified` as sentinel and resolve inside the composable.

Use the Color.Unspecified sentinel pattern:
```kotlin
@Composable
public fun AeroBadge(
    text: String,
    modifier: Modifier = Modifier,
    color: Color = Color.Unspecified,
    contentColor: Color = Color.Unspecified
) {
    val resolvedBg = if (color == Color.Unspecified) AeroTheme.colors.primary else color
    val resolvedFg = if (contentColor == Color.Unspecified) AeroTheme.colors.onPrimary else contentColor
    Box(
        modifier
            .clip(RoundedCornerShape(50))
            .background(resolvedBg)
            .heightIn(min = 16.dp)
            .padding(horizontal = 6.dp, vertical = 2.dp),
        contentAlignment = Alignment.Center
    ) {
        Text(text, color = resolvedFg, style = AeroTheme.typography.label)
    }
}
```

import androidx.compose.foundation.layout.*
import androidx.compose.material3.Text

@Composable
fun RangeSection() {
    val colors = AeroTheme.colors
    val typography = AeroTheme.typography
